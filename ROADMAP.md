# GRIMOIRE Roadmap

## Now — v0.1.0 (June 2026)
- 15 spells across 4 categories
- 8 CLI commands (list, search, install, cast, create, validate, publish, init)
- GDL schema validation (Zod-based)
- Cross-platform (macOS, Linux, Windows via Node.js)

## Next — v0.2.0 (July 2026)
- Spell composition: chain multiple spells in sequence (output of spell A -> input of spell B)
- Spell discovery: search by tags, ratings, and popularity
- `grimoire update <spell>`: update installed spells to latest version
- `grimoire uninstall <spell>`: remove installed spells

## v0.3.0 (August 2026)
- Web-based spell browser (Vue/React UI)
- Community spell ratings and reviews
- Spell dependencies (spell A requires spell B)

## v0.4.0 (September 2026)
- VSCode extension for inline spell casting
- Spell marketplace with creator payouts
- 50+ community-contributed spells

## v1.0.0 (2026)
- Stable GDL specification
- Spell composition with typed piping
- Official spell library: 100+ curated spells
- Documentation site with spell explorer
- CI/CD spell validation pipeline

## Ideas (future)
- Spell testing framework (unit tests for spells)
- Local spell execution via LOCUS AI (no external API needed)
- Spell-to-API bridge (wrap any REST API as a GDL spell)
- Community spell contests with prizes
- Enterprise spell registry (private org spells)
